## RGB거리

#### 점화식 정의하기

* d[N] = N번집까지 칠하는 비용의 최솟값

#### 제한조건 적용

* 이전 집과 현재 집의 색은 달라야 한다. 그러므로, 점화식을 변형한다. N번집을 RGB로 칠할때, 각 경우의 최소비용으로 재정의한다.

```
d[N][RGB] = N번집을 RGB로 칠할때, 각 경우의 최소비용
```

#### 점화식 세우기

```
d[N][R] = d[N][R] + MIN( d[N-1][G], d[n-1][B] )
나머지 G, B도 위와 비슷하게 세우면 된다.
```
